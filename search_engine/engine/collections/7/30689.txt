SQL Ganti Fungsi, tujuan, sintaks dan penggunaan umum. Fungsi SQL Replace memungkinkan kita untuk mencari frase karakter tertentu dalam sebuah string dan menggantinya dengan frase karakter lain. String yang diperbarui kemudian dikembalikan oleh fungsi. Sintaks untuk fungsi string ini sama untuk SQL Server, Oracle, dan Microsoft Access. Sintaksnya adalah sebagai berikut: EPLACE(stringToLookIn, stringToMatch, replacementsString). Sintaksnya cukup lurus ke depan, parameter stringToMatch adalah frasa karakter yang ingin kita ganti, replacementsString adalah frasa karakter yang akan menggantikan kemunculan parameter stringToMatch.
