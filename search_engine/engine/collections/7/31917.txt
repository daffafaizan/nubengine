Log transaksi pertukaran. Di Microsoft Exchange, log transaksi adalah file yang berisi catatan perubahan yang dibuat ke database Exchange. Informasi yang perlu ditambahkan ke database kotak surat pertama-tama ditulis ke log transaksi Exchange dan kemudian konten log transaksi tersebut kemudian ditulis ke database Exchange Server. Microsoft memilih untuk menggunakan model log transaksi sehingga pengguna masih dapat mengakses Exchange Server selama pencadangan.
