Tekan tombol Viera Link atau tombol Viera Tools pada remote control untuk mengakses kontrol Viera Link. Kontrol Viera Link muncul di layar.Gunakan tombol navigasi untuk memilih komponen, seperti Perekam, Camcorder, Player atau Home Theater..Fungsi Viera Link harus diaktifkan sebelum komponennya dapat dikontrol oleh remote. Tekan tombol Menu pada remote control untuk menampilkan menu utama. Gunakan tombol navigasi untuk memilih Setup, Viera Link Setting dan Viera Link. Gunakan tombol navigasi untuk mengatur Viera Link ke On..
