Dengan Aplikasi Editor HTML ini, pengguna dapat membuat dokumen HTML dan menyimpannya ke disk dalam format HTML atau teks. Jika disimpan sebagai file .htm, dokumen dapat dilihat di Windows Internet Explorer atau di Aplikasi Editor HTML. File yang disimpan sebagai .txt dapat dilihat di editor teks apa pun. Tutorial ini menunjukkan bagaimana melakukan hal berikut: 1 Membuat dan menggunakan daerah yang dapat diedit dengan atribut CONTENTEDITABLE.
