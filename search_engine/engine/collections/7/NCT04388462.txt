
    
      FE analyses will be performed using the finite element code COSMOS/M. The three-dimensional
      "lower leg" FE model (LL FE model) will be used included the tibia and the fibula. The bony
      structures will be generated by segmentation of a data set of computed tomographies from the
      Visible Human Project.
