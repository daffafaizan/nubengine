Apa itu format file IMG? Format file IMG adalah salah satu file berikut: 1. File gambar CD atau DVD, pada dasarnya setara dengan file ISO. Pada file seperti itu, cukup mengubah ekstensi dari IMG ke ISO dapat membuatnya dapat digunakan sebagai yang terakhir oleh sebagian besar program. 2.Sebuah format arsip yang digunakan untuk membuat image disk dari floppy disk dan HDD.
