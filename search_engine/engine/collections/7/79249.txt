PTB adalah singkatan dari Power Tab File. File yang berisi ekstensi file .ptb paling sering menyimpan tabulasi musik yang digunakan dalam pembuatan lembaran musik untuk instrumen berdawai seperti bass dan gitar. Banyak file tablature yang berisi akhiran file .ptb dibuat dengan perangkat lunak pembuat tablature Power Tab Editor. Software Peachtree Corporate Accounting juga diketahui menggunakan akhiran file .ptb. Ketika file PTB dikaitkan dengan perangkat lunak Peachtree, file tersebut berisi file data cadangan yang digunakan oleh aplikasi untuk memulihkan data yang hilang atau rusak.
