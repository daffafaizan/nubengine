Sekarang, bagaimana dengan partikulat dalam nyala api? Sementara itu jelas akan tergantung pada apa yang terbakar, saya membayangkan bahwa kolom api 10m dari pipa pelepasan gas di anjungan minyak akan beratnya kurang dari kolom api 10m dari, katakanlah, hutan pinus naik.o, karena udara (di permukaan laut) berbobot sekitar 1,3 kg per meter kubik (1,3 gram per liter), api berbobot sekitar 0,3 kg per meter kubik. Satu pon api biasa, di sini di Bumi dekat permukaan laut, akan memakan sebuah kubus sekitar 1,2 meter ke samping.
