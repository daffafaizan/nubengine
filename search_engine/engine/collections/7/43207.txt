Singkatnya, parsing XML berarti bahwa sebuah program menghasilkan representasi internal dari XML serial. Sebagai contoh, mari kita gunakan HTML. Parsing HTML pada dasarnya sama dengan parsing XML (kecuali sintaksnya sedikit berbeda, tapi itu tidak masalah sekarang). Seperti yang mungkin Anda ketahui, Anda dapat membuka dokumen XML dan HTML di editor teks apa pun dan melihat kontennya. Anda akan melihat semua elemen diserialisasi menggunakan tag pembuka dan penutup. Dalam HTML, sebuah paragraf dibuka menggunakan <p> dan ditutup menggunakan </p>. Segala sesuatu di antara dua tag itu adalah isi paragraf.
