apa itu file .ddc? File DivX Descriptor diklasifikasikan sebagai file data, dan perangkat lunak DivX Descriptor menggunakan file yang disimpan dalam format DDC sebagai file DivX Descriptor. File DDC ini ditambahkan dengan ekstensi .ddc, dan Daren Softwares mengembangkan format file DDC, bersama dengan aplikasi DivX Descriptor. Data yang disimpan dalam file DDC ini mencakup deskripsi file video berkode DVD yang terintegrasi dan dikompresi menggunakan spesifikasi DivX.
