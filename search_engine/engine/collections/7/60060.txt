Jika Anda menggunakan Alat Pemadam Kimia Kering biasa dan bukan Alat Pemadam Kimia Kering serbaguna, gunakan hanya pada kebakaran Kelas B dan Kelas C. Penting untuk menggunakan alat pemadam yang tepat untuk jenis bahan bakar karena menggunakan alat pemadam api yang salah dapat lebih berbahaya daripada baik dengan menyalakan kembali api. Bahan Kimia Basah. Alat pemadam busa hanya dapat digunakan pada kebakaran Kelas A dan Kelas B saja. Jika digunakan pada kebakaran Kelas C, akan menimbulkan bahaya kejut. Karbon dioksida. Alat pemadam api jenis ini mengambil oksigen dari api dan menghilangkan panas dengan debit dingin.
