Menyimpan. File IGES digunakan oleh program Computer Aided Design (CAD). File-file ini berisi desain gambar dan model dua dimensi dan tiga dimensi. Menurut Institut Nasional Standar dan Teknologi, format IGES memungkinkan desain CAD untuk ditransfer ke sistem yang berbeda. Perangkat lunak CAD standar, seperti Autodesk's AutoCAD dapat membuka file-file ini, tetapi jika Anda hanya ingin melihat file dan sedikit lainnya, Anda dapat menggunakan penampil gratis seperti IGS Viewer.
