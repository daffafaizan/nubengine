Detail Konstruktor StringBuffer public StringBuffer() Membangun buffer string tanpa karakter di dalamnya dan kapasitas awal 16 karakter. StringBuffer public StringBuffer(int capacity) Membangun buffer string tanpa karakter di dalamnya dan kapasitas awal yang ditentukan. StringBuffer(String str) Membangun buffer string yang diinisialisasi ke konten string yang ditentukan. append(boolean b) Menambahkan representasi string dari argumen boolean ke urutan.
