Cara memainkan file VOB / IFO / BUP. Jika Anda mendapatkan banyak file termasuk file IFO, VOB dan BUP, maka Anda memiliki file DVD-Video. Ini adalah file langsung dari film DVD. Dapat dimainkan di berbagai pemutar langsung dari HDD Anda, tetapi apa sebenarnya itu? VOB adalah singkatan dari Objek Video DVD. Ini pada dasarnya adalah salah satu file inti yang ditemukan pada disk DVD-Video dan berisi data film yang sebenarnya, termasuk aliran video MPEG-2 dan audio pendukung (MP2, AC3, DTS, dll.) Untuk melihat file VOB yang berdiri sendiri, Anda harus memiliki perangkat lunak pemutar DVD yang mendukung pemutaran VOB. IFO.
