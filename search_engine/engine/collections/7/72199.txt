Seseorang dapat meminjamkan atau menyewakan senjata api kepada penduduk Negara Bagian mana pun untuk penggunaan sementara untuk tujuan olahraga yang sah, jika dia tidak mengetahui atau memiliki alasan yang masuk akal untuk meyakini bahwa orang tersebut dilarang menerima atau memiliki senjata api berdasarkan undang-undang Federal. Seseorang dapat menjual atau mentransfer senjata api kepada penerima lisensi di Negara Bagian mana pun.
