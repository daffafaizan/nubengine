Dapatkan Jalur Direktori dari file Batch yang dieksekusi. Kebanyakan orang mungkin tahu bahwa dapat menggunakan variabel %0 dalam file batch untuk mendapatkan nama file batch yang dieksekusi. Namun jika Anda menggunakannya bersama dengan beberapa pengubah parameter batch, Anda dapat dengan mudah membaginya menjadi drive, direktori, atau nama file. Karena file instal biasanya berada di direktori yang sama dengan file batch, saya dapat menggunakan %~dp0 sebagai jalurnya. Sekarang ketika saya mengklik dua kali pada file batch di windows explorer apakah drive dipetakan atau jalur UNC, file batch memiliki jalur yang benar ke file.
