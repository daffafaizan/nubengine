Singkatnya, Anda perlu mengunduh program bernama Odin 3 dan perangkat lunak rooting yang disebut CF-Root-SGS3-v6.4. Hubungkan telepon Anda ke PC, buat cadangannya dan instal driver terbaru menggunakan perangkat lunak Kies Samsung, lalu putuskan sambungannya. Ekstrak dan instal Odin lalu ekstrak file CF-Root, memberi Anda file .TAR.
