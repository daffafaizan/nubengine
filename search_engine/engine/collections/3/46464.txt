Sebelum saya mulai, berikut adalah cara kami biasanya menghapus kata sandi dari dokumen yang dilindungi kata sandi di Microsoft Word. Setelah membuka dokumen di Word, buka Save As, dan klik Tools di jendela Save As, pilih General Option. Dan hapus kata sandi dari kotak Kata Sandi untuk Memodifikasi, klik OK, dan simpan dokumen. Ikuti langkah-langkah berikut: 1. Ubah nama ekstensi nama dokumen menjadi ZIP, seperti file terkompresi ZIP. 2. Klik dua kali dan buka di Windows Explorer, arahkan ke folder Word. 3. Hapus file Settings.xml dari folder tersebut. Dan kemudian tutup Jendela. 4. Ganti nama kembali ke file DOCX. 5. Dan selesai.
