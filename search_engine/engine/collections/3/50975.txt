Mikrofon MEMS Perangkat Mikrofon - terintegrasi dalam satu chip - dibuat menggunakan teknologi CMOS. Teknologi sebelumnya : mikrofon kondensor electret (ECS) - dirakit dari bagian-bagian terpisah. Jenis Mikrofon MEMS Mikrofon MEMS Analog mengubah input tekanan suara menjadi output tegangan analog. Mikrofon MEMS Digital mengubah input tekanan suara menjadi sinyal output digital, biasanya dalam format modulasi kepadatan pulsa (PDM). Prinsip Mikrofon MEMS - Ada tegangan antara membran dan pelat belakang.
