Cara mendapatkan nomor id pajak federal. Nomor FEIN dan nomor EIN sama dengan apa yang disebut sebagai nomor id pajak federal, atau nomor identifikasi pemberi kerja. Nomor id pajak diperoleh dengan IRS. Ada beberapa cara Anda bisa mendapatkannya. Anda dapat melakukannya secara online di situs web IRS, Anda dapat mengirimkan file faks, Anda dapat menghubungi IRS secara langsung, Anda dapat mengirimkan aplikasi melalui pos, atau Anda dapat menyewa layanan perolehan nomor FEIN profesional seperti Agen Terdaftar Northwest.
