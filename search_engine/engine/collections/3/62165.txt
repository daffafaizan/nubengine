Dalam penginstalan baru Windows 10 untuk banyak pengguna yang memenuhi syarat, pengaturan default adalah menyimpan file ke OneDrive, bukan ke PC Anda. Ini berarti Anda dapat mengakses file tersebut dari perangkat apa pun yang dapat terhubung ke Internet, dan file tersebut dilindungi jika komputer Anda rusak.
