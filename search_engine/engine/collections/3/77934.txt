NVL-Tanah. NVL hanya tersedia di Oracle, dan tidak di MySQL atau SQL Server. Tidak seperti banyak istilah pemrograman/database, NVL bukanlah akronim untuk apa pun. Ini hanya NVL, meskipun dapat membantu untuk menganggapnya sebagai Nilai Null. NVL adalah fungsi substitusi; yaitu, digunakan untuk menampilkan satu nilai jika nilai lain adalah NULL. Dan bukan hanya nol, tetapi NULL, kosong, batal. Pikirkan tabel data karyawan: Jika bidang data tertentu adalah NULL, seperti nomor ponsel, Anda dapat menggunakan NVL untuk menggantikan nomor generik (mungkin telepon meja mereka, atau 999-9999).
