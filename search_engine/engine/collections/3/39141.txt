Gambar Grafik Jaringan Portabel. File PNG adalah Gambar Grafik Jaringan Portabel. PNG (Portable Network Graphics) adalah format gambar bitmap terkompresi tanpa kehilangan. PNG diciptakan untuk memperbaiki dan mengganti format GIF dengan format file gambar yang tidak memerlukan lisensi paten untuk digunakan. Deskripsi format PNG. Kategori: File grafis.
