Program Pencegahan Obesitas Program Pencegahan Obesitas Program Pencegahan Obesitas bekerja untuk membuat pilihan sehat lebih mudah bagi semua orang Texas di mana pun mereka tinggal, bekerja, dan bermain. Program Pencegahan Obesitas dibuat pada Oktober 2013 dengan menggabungkan Nutrisi, Aktivitas Fisik dan Pencegahan Obesitas (NPAOP) dan Program Kesehatan Tempat Kerja.
