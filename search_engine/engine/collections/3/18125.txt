Handel - Musik untuk Kembang Api Kerajaan. Musik gembira Handel untuk Kembang Api Kerajaan adalah musik yang cocok untuk seorang raja - George II, pada kenyataannya, yang menghadiri pertunjukan pertama di Sungai Thames pada tahun 1749. Handel menyusun Musiknya untuk Kembang Api Kerajaan saat bekerja untuk George II pada tahun 1749. Perjanjian dari Aix-La-Chapelle ditandatangani untuk menandai berakhirnya Perang suksesi Austria dan karya itu disusun untuk mengiringi perayaan kerajaan.
