Bagian berikut ini Anda dapat menemukan cara membuka dan mengedit lembar kerja Excel melalui VB.NET. Untuk membuka atau mengedit lembar kerja Excel di VB.NET, Anda harus menambahkan Perpustakaan Objek Microsoft Excel 12.0 di proyek Anda. Dari gambar berikut untuk menunjukkan cara menambahkan pustaka referensi Excel di proyek Anda.1. Buat proyek baru dan tambahkan tombol ke Formulir. Pilih dialog referensi dari menu Proyek. Pilih Perpustakaan Objek Microsoft Excel 12.0 dan klik tombol OK. Sekarang Anda dapat mulai coding untuk membuka atau membaca dari file Excel dan mengedit sel..
