MDI adalah singkatan dari Microsoft Document Imaging format. Format file berbasis TIFF ini dirancang untuk menyimpan gambar menurut tata letak halaman menggunakan Office Document Imaging, yang dapat menyimpan file sebagai MDI dan TIFF. File TIFF kompatibel secara luas, tetapi file MDI hanya terbuka dengan Office Document Imaging, yang dapat menjadi masalah ketika Anda harus berbagi file MDI. Solusi terbaik adalah menyimpannya sebagai TIFF untuk memulai, tetapi itu tidak banyak membantu jika Anda yang mencoba membuka file MDI tanpa Office. MDI Viewer by HAN Software membuka dan menampilkan gambar MDI dan TIFF tanpa harus memiliki Office, Adobe Acrobat, atau barang besar lainnya.
