Cara memulai exe dalam file batch Untuk memulai file exe dari file batch di Windows, Anda dapat menggunakan perintah start. Misalnya, baris berikut akan memulai Notepad di sebagian besar versi Windows. MULAI C:\Windows\NOTEPAD.EXE.Perintah start dapat digunakan untuk file exe lainnya dengan mengganti jalur file dengan jalur ke file exe.Tip: Jika Anda ingin memulai beberapa file yang dapat dieksekusi, ingatlah bahwa file batch membuka setiap file segera. atau contoh, baris berikut akan memulai Notepad di sebagian besar versi Windows. MULAI C:\Windows\NOTEPAD.EXE. Perintah start dapat digunakan untuk file exe lainnya dengan mengganti path file dengan path ke file exe.
