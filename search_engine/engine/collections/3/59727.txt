RINGKASAN. Dalam beberapa keadaan, database Oracle Anda dapat menghasilkan file .trc saat digunakan dengan Microsoft Transaction Server. Dokumen ini menjelaskan apa itu file TRC, mengapa file TRC diproduksi, dan apa yang dapat Anda lakukan untuk menghilangkan file TRC. File TRC dihasilkan ketika MS DTC mencoba melakukan operasi XA dengan database Oracle Anda dan operasi Oracle XA melaporkan kesalahan. File TRC adalah sarana yang digunakan Oracle untuk melaporkan informasi kesalahan tambahan.
