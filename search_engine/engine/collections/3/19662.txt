Operator asuransi menggunakan gaji tahunan untuk menghitung premi untuk manfaat berbasis pendapatan seperti LTD, STD dan asuransi jiwa. Untuk karyawan per jam, kami akan secara otomatis menghitung gaji pokok tahunan berdasarkan Jam per Minggu x Tarif Per Jam x 52 minggu. Untuk gaji pegawai, ini merupakan input nilai sebagai Gaji Pokok Tahunan. Untuk kedua jenis karyawan, jika nilai ini tidak digunakan oleh operator asuransi untuk menghitung premi untuk manfaat berbasis pendapatan, bidang baru telah dibuat yang disebut Gaji Manfaat Tahunan.
