https://www.gnu.org/software/wget/. Wget adalah utilitas gratis untuk mengunduh file non-interaktif dari Web. Ini mendukung protokol HTTP, HTTPS, dan FTP, serta pengambilan melalui proxy HTTP. Itu dapat mengikuti tautan di halaman HTML dan membuat versi lokal dari situs web jarak jauh, sepenuhnya menciptakan kembali struktur direktori situs asli.
