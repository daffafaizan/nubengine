Navigasi semacam ini juga didukung oleh browser web berbasis Mozilla, yang disebut sebagai penjelajahan caret. Tombol pintas (F7) ditetapkan untuk beralih antara mode caret dan mode non-caret. Dalam mode tanda sisipan, tanda sisipan yang berkedip muncul di dalam halaman web. Pengguna kemudian dapat menavigasi halaman web menggunakan tombol panah.
