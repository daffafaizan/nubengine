Semua router dan gateway Internet Kecepatan Tinggi berasal dari pabrikan dengan nama pengguna dan kata sandi default. Setiap produsen dapat menggunakan kombinasi nama pengguna dan kata sandi yang berbeda sebagai default. Jika Anda tidak mengubah kata sandi default saat mengatur perute, Anda dapat menggunakan tabel di bawah ini untuk mencari nama pengguna dan kata sandi default untuk perute atau gateway Anda.
