Jika file QHS Anda tidak terbuka, kemungkinan besar Anda tidak menginstal aplikasi perangkat lunak yang benar pada PC Anda untuk melihat atau mengedit ekstensi file QHS. Jika PC Anda membuka file QHS, tetapi itu adalah aplikasi yang salah, Anda harus mengubah pengaturan asosiasi file registri Windows Anda. Dengan kata lain, Windows mengaitkan ekstensi file QHS dengan program perangkat lunak yang salah.
