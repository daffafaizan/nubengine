Tapi serius, Anda menandai pertanyaan dengan 'bahasa pemrograman'. Java adalah bahasa pemrograman seperti yang lain. Seseorang dapat menggunakannya untuk menulis program komputer. Java juga dapat merujuk ke Java Runtime Environment atau JRE singkatnya, kumpulan perangkat lunak yang diperlukan untuk menjalankan program yang ditulis dalam bahasa Java. Ini mungkin yang Anda 'unduh'.
