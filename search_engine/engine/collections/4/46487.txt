Menjalankan Disk Defragmenter di Windows 7 memungkinkan komputer Anda untuk mengatur ulang semua data yang terfragmentasi, yang, pada gilirannya, dapat meningkatkan kecepatan dan efisiensi komputer Anda secara keseluruhan. Di Windows 7, Anda dapat mendefrag komputer Anda secara manual kapan saja, atau mengatur jadwal defrag reguler menggunakan Disk Defragmenter. Ikuti petunjuk ini untuk mendefrag komputer Windows 7 Anda. Klik Disk Defrag atau Defragment Now untuk memulai proses defrag. Komputer Anda akan memakan waktu antara beberapa menit dan beberapa jam untuk mendefrag drive tergantung pada ukurannya dan status terfragmentasi saat ini.
