Registry Windows, biasanya disebut hanya sebagai registry, adalah kumpulan database pengaturan konfigurasi di sistem operasi Microsoft Windows. Registry Windows terkadang salah dieja sebagai registery atau register. Registry Windows digunakan untuk menyimpan banyak informasi dan pengaturan untuk program perangkat lunak, perangkat keras, preferensi pengguna, konfigurasi sistem operasi, dan banyak lagi.
