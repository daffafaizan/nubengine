Platform yang Didukung Wunderlist memiliki aplikasi desktop untuk Mac dan Windows, ditambah aplikasi seluler untuk Android, iOS, dan Windows Phone, tetapi tidak untuk BlackBerry. Ada juga plugin browser untuk Chrome, Firefox, dan Safari. Todoist memiliki aplikasi desktop, serta aplikasi Android dan iOS, tetapi belum ada untuk Windows Phone. Todoist juga memiliki plug-in untuk Chrome dan Firefox (tetapi bukan Safari), meskipun ia mengalahkan Wunderlist dengan menawarkan integrasi email untuk Gmail, Outlook, dan Thunderbird.
