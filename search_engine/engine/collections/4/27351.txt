Ekstensi File Microsoft Word 2010. Ekstensi File Di Word 2010. Microsoft Word 2010 menggunakan ekstensi file .docx untuk menyimpan dokumen. Namun, dokumen yang memiliki ekstensi file .docx belum tentu dokumen Word 2010; itu mungkin juga dokumen Word 2007 karena itu adalah ekstensi yang digunakan oleh rilis Word itu.
