Punya thumbs.db di Windows Explorer? Inilah mereka dan apa yang dapat Anda lakukan dengan aman dengan mereka. Saya menghabiskan banyak waktu di Windows Explorer melakukan berbagai tugas. Sesekali saya menemukan file yang tidak saya kenal dan saya tidak yakin apa yang harus saya lakukan. Thumbs.db adalah contoh yang baik, ia memiliki ekstensi aneh: .db. Anda tidak akan melihat file thumbs.db kecuali Anda telah mencentang Tampilkan File dan Folder Tersembunyi di panel Opsi Folder dan menggunakan mode ikon di Explorer, jadi jika Anda belum melihatnya, mungkin itulah alasannya. :) Thumbs.db sangat mirip dengan namanya.
