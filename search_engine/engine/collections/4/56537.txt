Hapus Host Terkelola dari Server vCenter di Klien Web vSphere. Anda dapat menghapus host yang dikelola dari vCenter Server untuk menghentikan vCenter Server dari pemantauan dan pengelolaan host tersebut.Jika memungkinkan, hapus host yang dikelola saat mereka terhubung.Center Server menghapus host yang dikelola dan mesin virtualnya dari inventaris. vCenter Server kemudian membebaskan jumlah kapasitas lisensi yang dialokasikan untuk host. Tuan rumah mempertahankan kunci lisensi yang ditetapkan.
