GNU Wget (atau hanya Wget, sebelumnya Geturl) adalah program komputer yang mengambil konten dari server web, dan merupakan bagian dari Proyek GNU. Namanya berasal dari World Wide Web dan get. Ini mendukung pengunduhan melalui protokol HTTP, HTTPS, dan FTP. Unduhan ecursive juga berfungsi dengan FTP, di mana Wget mengeluarkan perintah LIST untuk menemukan file tambahan mana yang akan diunduh, mengulangi proses ini untuk direktori dan file di bawah yang ditentukan di URL teratas. Wildcard seperti shell didukung saat pengunduhan URL FTP diminta.
