Berlaku Untuk: Windows PowerShell 2.0, Windows PowerShell 3.0, Windows PowerShell 4.0, Windows PowerShell 5.0. Bagian ini berisi topik bantuan untuk modul Microsoft.PowerShell.Host. Modul Microsoft.PowerShell.Host berisi cmdlet yang mengelola fitur program yang menghosting Windows PowerShell. Itu diinstal dengan Windows PowerShell.
