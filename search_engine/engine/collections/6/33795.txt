Apa itu tornado api? Tornado api atau pusaran api hanyalah tornado yang terbuat dari api. Itu terjadi ketika kondisi tertentu (tergantung pada suhu dan arus udara), memperoleh vortisitas vertikal dan membentuk pusaran, atau efek seperti tornado. Tornado api dapat berupa angin puyuh yang terpisah dari api, baik di dalam area terbakar maupun di luarnya.
