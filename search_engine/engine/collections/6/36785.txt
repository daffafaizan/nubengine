Ketik perintah berikut untuk mengunduh satu halaman dari situs web: wget http://www.example.com/filename. Ganti www.example.com dengan nama situs yang ingin Anda unduh, dan ganti nama file dengan nama file yang ingin Anda unduh. Langkah 4.he perintah wget memungkinkan Anda untuk mengunduh file dari Internet menggunakan sistem operasi Linux seperti Ubuntu.
