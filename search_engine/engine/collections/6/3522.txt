JavaScript adalah bahasa pemrograman komputer yang ditafsirkan. Perbedaan utama antara Java dan JavaScript adalah bahwa sementara Java adalah bahasa pemrograman, JavaScript adalah bahasa skrip. Ini adalah bahasa skrip berbasis prototipe yang dinamis, diketik dengan lemah, dan memiliki fungsi kelas satu. Namun, seperti Java, JavaScript sangat dipengaruhi oleh bahasa pemrograman C.
