WMV adalah singkatan dari Windows Media Video. File .wmv adalah format video terkompresi yang dikembangkan oleh Microsoft untuk aplikasi streaming Internet. Banyak video dan film yang ditemukan online dalam format .wmv. Saat file .wmv diputar, file tersebut didekompresi oleh pembaca file saat sedang diputar ulang di komputer Anda. Meskipun file .wmv dapat dibaca oleh sebagian besar pemutar media, file tersebut dapat dikonversi ke format lain jika perlu.
