ALTON, TX (Kabupaten HILDAGO) (Kabupaten HIDALGO) ALTON, TEXAS (Kabupaten Hidalgo). Alton berjarak empat mil di utara Mission on State Highway 107 di Hidalgo County. Ini berasal sebagai perhentian di San Benito dan Rio Grande Valley (Spiderweb) Kereta Api pada tahun 1911, ketika populasi lima puluh, dan diberi nama oleh pejabat kereta api setelah Kereta Api Alton yang telah didirikan oleh warga Alton, Illinois. Sebuah kantor pos beroperasi di sana dari tahun 1913 hingga 1916. Pada akhir 1920-an, Distrik Sekolah Independen Alton dibentuk.
