Klik kanan pada pintasan file dan klik opsi Pin to Start. Itu dia. Ini akan segera menyematkan pintasan file ke Layar Mulai. CATATAN: Jika pintasan tidak muncul di layar Semua Aplikasi, coba pindahkan pintasan dari folder Start Menu ke folder Start Menu\Programs.
