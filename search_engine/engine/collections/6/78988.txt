Proses Windows Shell Experience Host mungkin menggunakan sedikit memori atau CPU saat dijalankan. Untuk mengetahui apakah ini yang terjadi pada mesin Anda yang menjalankan Windows 10, gunakan pintasan keyboard Ctrl-Shift-Esc untuk membuka Pengelola Tugas. Klik tautan lebih detail di kiri bawahnya dan temukan proses Windows Shell Experience Host di daftar. Anda dapat mengklik nama untuk mengurutkan proses menurut abjad yang dapat memudahkan untuk menemukannya dalam daftar.
