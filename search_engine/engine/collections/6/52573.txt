Cara Membuka Port NAT Secara Manual. Jika Anda menggunakan metode ini, pastikan xbox/PS3 Anda telah diberi alamat IP statis dari router. Berikut adalah contoh di mana melakukan ini pada router Netgear. Ketika Anda membuka port NAT, Anda perlu menentukan alamat IP untuk meneruskannya. Jika nanti alamat IP xbox/Ps3 berubah maka akan berhenti bekerja. Jika semuanya gagal, Anda dapat membuka semua port NAT dengan mengaktifkan opsi DMZ dari router NAT dan ketik alamat IP xbox 360/PS3 Anda. Perhatikan bahwa ini masih memerlukan alamat IP statis. Ini sebenarnya lebih mudah daripada membuka port NAT secara manual tetapi saya hanya akan merekomendasikannya sebagai upaya terakhir.
