Ekstensi PIT adalah salah satu format kompresi file Macintosh asli. Ini telah menjadi usang, dan digantikan oleh algoritma kompresi yang lebih canggih, seperti Zip dan Stuffit. File PIT dikategorikan sebagai file terkompresi dan dikenal sebagai Arsip Terkompresi PackIt karena sebagian besar digunakan oleh alat MAC OS PackIt. Namun, meskipun file PIT sangat berguna terutama dalam hal kompresi PackIt, file-file ini juga sudah menjadi file usang.
