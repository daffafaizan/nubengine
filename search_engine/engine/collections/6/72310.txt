File kode C++ (dengan ekstensi .cpp) bukan satu-satunya file yang biasa terlihat di program C++. Jenis file lainnya disebut file header, kadang-kadang dikenal sebagai file include. File header biasanya memiliki ekstensi .h, tetapi terkadang Anda akan melihatnya dengan ekstensi .hpp atau tanpa ekstensi sama sekali. Tujuan dari file header adalah untuk menyimpan deklarasi untuk file lain untuk digunakan.
