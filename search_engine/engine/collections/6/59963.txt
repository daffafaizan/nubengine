Pernyataan MERGE di SQL Server 2008. Ketika pernyataan SQL MERGE diperkenalkan di SQL Server 2008, itu memungkinkan pemrogram database untuk mengganti rim kode yang berantakan dengan sesuatu yang cepat, sederhana dan dapat dipelihara. Sintaks MERGE hanya membutuhkan sedikit penjelasan, dan Rob Sheldon, seperti biasa, siap menjelaskan dengan banyak contoh. Dimulai dengan SQL Server 2008, Anda dapat menggunakan pernyataan MERGE untuk mengubah data dalam tabel target berdasarkan data dalam tabel sumber. Pernyataan menggabungkan target ke sumber dengan menggunakan kolom umum untuk kedua tabel, seperti kunci utama.
