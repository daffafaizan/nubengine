Perintah nslookup digunakan untuk menanyakan server nama internet secara interaktif untuk mendapatkan informasi. Gambaran. nslookup, yang merupakan singkatan dari pencarian server nama, adalah alat yang berguna untuk menemukan informasi tentang domain bernama. Secara default, nslookup akan menerjemahkan nama domain ke alamat IP (atau sebaliknya). Misalnya, untuk mengetahui apa alamat IP microsoft.com, Anda dapat menjalankan perintah: nslookup microsoft.com
