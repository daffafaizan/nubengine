WPF, yang merupakan singkatan dari Windows Presentation Foundation, adalah pendekatan terbaru Microsoft untuk kerangka kerja GUI, digunakan dengan kerangka .NET. Tapi apa itu kerangka kerja GUI? GUI adalah singkatan dari Graphical User Interface, dan Anda mungkin sedang melihatnya sekarang. Windows memiliki GUI untuk bekerja dengan komputer Anda, dan browser tempat Anda mungkin membaca dokumen ini memiliki GUI yang memungkinkan Anda menjelajahi web. Kerangka kerja GUI memungkinkan Anda membuat aplikasi dengan berbagai elemen GUI, seperti label, kotak teks, dan elemen terkenal lainnya.
