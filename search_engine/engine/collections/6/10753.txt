Perusahaan: Microsoft. File: Thumbs.db. Peringkat Keamanan: Thumbs.db adalah file cache yang digunakan oleh Windows di folder mana pun yang menemukan gambar atau video. Hal ini dimaksudkan untuk mempercepat tampilan thumbnail. Windows Explorer menggunakan thumbs.db untuk melihat gambar cache saat folder dilihat dalam tampilan Thumbnail.
