JDK vs JRE. JDK adalah singkatan dari Java Developer Kit sedangkan JRE adalah singkatan dari Java Runtime Environment. JRE adalah apa yang dibutuhkan untuk menjalankan aplikasi Java. Dibundel di dalam adalah implementasi dari Java Virtual Machine (JVM). JDK menyertakan JRE dan banyak alat pengembangan aplikasi Java lainnya. Misalnya, kompiler Java (javac) hanya disertakan dalam JDK. Jika Anda ingin menjalankan perangkat lunak Java, Anda cukup tinggal dengan JRE.
