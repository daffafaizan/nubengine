Untuk mengatur penerusan port pada router iBall iB-LR6111A, perangkat Anda harus memiliki alamat IP statis. Anda harus menyelesaikan salah satu langkah berikut terlebih dahulu: 1 Coba program gratis kami yang akan menyiapkan alamat IP statis untuk Anda yang disebut PF Setup Static IP. Temukan alamat IP router Anda menggunakan panduan ini: Cara Menemukan Alamat IP Router Anda. 2 Atau Anda dapat menggunakan Perangkat Lunak Gratis kami untuk menemukan alamat IP router Anda. 3 PF Static IP Address dan PF Router Detector keduanya merupakan aplikasi perangkat lunak gratis yang akan membantu Anda menemukan alamat IP router Anda.
