editor HTML. Editor HTML adalah program komputer untuk mengedit HTML, markup halaman web. Meskipun markup HTML halaman web dapat ditulis dengan editor teks apa pun, editor HTML khusus dapat menawarkan kemudahan dan fungsionalitas tambahan. Misalnya, banyak editor HTML tidak hanya menangani HTML, tetapi juga teknologi terkait seperti CSS, XML dan JavaScript atau ECMAScript.
