Penampil VariCAD. Varicad Viewer adalah perangkat lunak penampil CAD gratis dan berguna untuk komputer Anda. Ini didasarkan pada VairCAD. Anda dapat dengan mudah membuka dan melihat file IGES (2D), Varicad, DXF (2D), DWG (2D), dan STEP (3D). Dengan menggunakan freeware ini Anda juga dapat mengonversi format file DXF ke DWG dan format file DWG ke DXF.
