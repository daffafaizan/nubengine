Jika file MNI Anda tidak terbuka, kemungkinan besar Anda tidak menginstal aplikasi perangkat lunak yang benar di PC Anda untuk melihat atau mengedit ekstensi file MNI. Jika PC Anda membuka file MNI, tetapi itu adalah aplikasi yang salah, Anda harus mengubah pengaturan asosiasi file registri Windows Anda. Dengan kata lain, Windows mengaitkan ekstensi file MNI dengan program perangkat lunak yang salah.
