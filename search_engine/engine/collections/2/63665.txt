Ini bukan editor HTML terbaik, tapi bagus untuk harganya, terutama jika Anda membangun dalam XML. Saya menggunakan Komodo Edit setiap hari untuk pekerjaan saya di XML dan saya sering menggunakannya untuk pengeditan HTML dasar juga. Ini adalah salah satu editor saya akan hilang tanpa. Ada dua versi Komodo: Komodo Edit dan Komodo IDE.
