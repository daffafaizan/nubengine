Apa itu OpenCV? OpenCV adalah perpustakaan open source terkemuka untuk visi komputer, pemrosesan gambar, dan pembelajaran mesin, dan sekarang menampilkan akselerasi GPU untuk operasi waktu nyata. OpenCV dirilis di bawah lisensi BSD dan karenanya gratis untuk penggunaan akademis dan komersial. Ini memiliki antarmuka C++, C, Python dan Java dan mendukung Windows, Linux, Mac OS, iOS dan Android.
