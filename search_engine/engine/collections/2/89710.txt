Pilih file atau folder yang akan dikompresi saat menggunakan sistem operasi Mac OS. Klik kanan pada file atau folder yang dipilih dan kemudian klik opsi Buat Arsip untuk mengompres file atau folder. Perhatikan bahwa Mac OS versi lama mungkin harus menggunakan perangkat lunak kompresi. Cari cara mengompresi file di Mac di mesin pencari untuk menemukan perangkat lunak yang kompatibel. Klik kanan pada file yang akan dikompresi saat menggunakan Windows Vista atau Windows 7. Arahkan kursor ke atas Kirim ke dan kemudian klik opsi folder Terkompresi (zip). Komputer akan secara otomatis membuat folder untuk menyimpan file dan mengompresnya.
