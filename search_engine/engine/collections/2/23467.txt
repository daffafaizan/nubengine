Untuk mengubah ukuran bilah tugas desktop Windows Anda, pastikan bilah tugas tidak terkunci. Untuk melihat apakah terkunci atau tidak, klik kanan ruang kosong di taskbar, dan centang tidak ada centang di sebelah 'Kunci taskbar'. Jika tidak ada, bagus. Jika ada, klik 'Lock the taskbar' sekali untuk membukanya. Jika ada, ubah ukuran taskbar desktop Windows Anda, pastikan taskbar tidak terkunci. Untuk melihat apakah terkunci atau tidak, klik kanan ruang kosong di bilah tugas, dan centang tidak ada centang di sebelah 'Kunci bilah tugas'. Jika tidak ada, bagus. Jika ada, klik 'Kunci bilah tugas' sekali untuk membukanya. Iklan.
