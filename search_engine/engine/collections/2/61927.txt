Definisi dan Penggunaan. Deklarasi <!DOCTYPE> harus menjadi hal pertama dalam dokumen HTML Anda, sebelum tag <html>. Deklarasi <!DOCTYPE> bukan tag HTML; itu adalah instruksi ke browser web tentang versi HTML apa halaman itu ditulis. Dalam HTML 4.01, deklarasi <!DOCTYPE> mengacu pada DTD, karena HTML 4.01 didasarkan pada SGML. DTD menentukan aturan untuk bahasa markup, sehingga browser merender konten dengan benar. HTML5 tidak didasarkan pada SGML, dan oleh karena itu tidak memerlukan referensi ke DTD.
