Oleh karena itu untuk mendapatkan lokasi file batch yang dieksekusi dari dalam file batch tersebut, Anda dapat menggunakan variabel %~dp0. Di mana d adalah drive, p adalah jalur dan 0 tentu saja adalah nama file batch yang dieksekusi. Ini sangat berguna bagi saya karena saya memiliki beberapa file batch pada drive jaringan yang melakukan beberapa instalasi sederhana. Karena file instalasi biasanya berada di direktori yang sama dengan file batch, saya dapat menggunakan %~dp0 sebagai jalurnya. Sekarang ketika saya mengklik dua kali pada file batch di windows explorer apakah drive dipetakan atau jalur UNC, file batch memiliki jalur yang benar ke file.
