Solusi: Atur Ulang Kata Sandi Router ke Pengaturan Pabrik. Melakukan hard reset pada router Anda akan mengatur ulang router Anda ke pengaturan default pabrik. Ini berarti bahwa nama pengguna dan kata sandi router juga akan diatur ulang ke pengaturan default. Setelah hard reset router Anda, Anda seharusnya dapat masuk menggunakan nama pengguna dan kata sandi default untuk router Anda.
