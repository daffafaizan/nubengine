Unduh Versi Windows Unduh Versi Mac. 1. Tambahkan file WMV. Setelah meluncurkan program, klik Tambah File untuk mengunggah file WMV ke jendela tampilan. Konversi batch didukung. 2. Atur format keluaran. Untuk memutar file WMV di Mac dengan sukses, Anda cukup memilih MOV sebagai format dari Profile > QuickTime Movie Format. Catatan: Aplikasi ini menyertakan sekelompok preset yang menyederhanakan proses pemilihan pengaturan output. Jika Anda ingin mengonversi file WMV secara eksklusif untuk salah satu perangkat yang tercantum dalam daftar prasetel, cukup klik perangkat tersebut. 3.
