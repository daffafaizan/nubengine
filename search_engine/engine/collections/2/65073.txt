Apa itu Api Kembar? Api kembar, juga disebut jiwa kembar, secara harfiah adalah bagian lain dari jiwa kita. Kita masing-masing hanya memiliki satu saudara kembar, dan umumnya setelah berpisah, keduanya berpisah, menjelma berulang-ulang untuk mengumpulkan pengalaman manusia sebelum kembali bersama. Idealnya, ini terjadi di kedua kehidupan terakhir mereka di planet ini sehingga mereka bisa naik bersama.
