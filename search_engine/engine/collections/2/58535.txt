Untuk memeriksa apakah JavaScript diaktifkan di browser Anda, klik tautan Periksa JavaScript di bawah. Jika Anda tidak melihat kotak popup yang mengatakan JavaScript diaktifkan, berarti Anda tidak mengaktifkan JavaScript di browser Anda. Periksa apakah JavaScript diaktifkan di browser ini. Jika Anda tidak mengaktifkan JavaScript, ikuti petunjuk di bawah ini yang berlaku untuk browser yang sedang Anda gunakan. Setelah mengaktifkan JavaScript, ingatlah untuk REFRESH atau RELOAD halaman ini sebelum mengklik tautan Periksa JavaScript di atas untuk memastikan bahwa JavaScript berfungsi.
