File Oracle XA TRC tipikal berisi informasi seperti yang ditunjukkan berikut ini. File TRC dihasilkan ketika MS DTC mencoba melakukan operasi XA dengan database Oracle Anda dan operasi Oracle XA melaporkan kesalahan. File TRC adalah sarana yang digunakan Oracle untuk melaporkan informasi kesalahan tambahan.
