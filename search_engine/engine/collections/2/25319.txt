Atur program default Anda. Gunakan opsi ini untuk memilih program mana yang Anda ingin Windows gunakan, secara default. Jika sebuah program tidak muncul dalam daftar, Anda dapat menjadikan program tersebut sebagai default dengan menggunakan Asosiasi Set. Buka Program Default dengan mengklik tombol Start dan kemudian mengklik Program Default. Klik Kaitkan jenis file atau protokol dengan program. Klik jenis file atau protokol yang Anda inginkan agar program bertindak sebagai default. Klik Ubah program. Klik program yang ingin Anda gunakan sebagai default untuk jenis file yang Anda pilih, atau klik panah di samping Program Lain untuk menampilkan program tambahan. (Jika Anda tidak melihat Program Lain, atau program Anda tidak terdaftar, klik Telusuri untuk menemukan program yang ingin Anda gunakan, lalu klik Buka.
