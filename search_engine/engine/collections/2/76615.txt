Proses yang dikenal sebagai Opc.Ua.DiscoveryServer milik perangkat lunak OPC UA SDK (versi 1.01 Redistributables, 1.1 Redistributables) atau OPC UA Local Discovery Server oleh OPC Foundation (www.opcfoundation.org). Deskripsi: Opc.Ua.DiscoveryServer.exe tidak penting untuk OS Windows dan menyebabkan masalah yang relatif sedikit. Opc.Ua.DiscoveryServer.exe terletak di subfolder C:\Program Files\Common Files. Ukuran file pada Windows 10/8/7/XP adalah 28.160 byte.
