SQL Server: Fungsi LEAD. Tutorial SQL Server ini menjelaskan cara menggunakan fungsi LEAD di SQL Server (Transact-SQL) dengan sintaks dan contoh. Keterangan. Di SQL Server (Transact-SQL), fungsi LEAD adalah fungsi analitik yang memungkinkan Anda membuat kueri lebih dari satu baris dalam tabel sekaligus tanpa harus menggabungkan tabel itu sendiri. Ini mengembalikan nilai dari baris berikutnya dalam tabel. Untuk mengembalikan nilai dari baris sebelumnya, coba gunakan fungsi LAG.
