Untuk menggunakan refactoring, Anda dapat menempatkan tanda sisipan: 1 Pada simbol di editor teks. 2 Pada pemilihan kode di editor teks. 3 Pada file atau pilihan file di Solution Explorer. 4 Pada jenis anggota atau pilihan jenis anggota dalam Struktur File atau jendela alat ReSharper lainnya.afe Hapus. Jika Anda akan menghapus jenis, anggota jenis, atau simbol lain dalam file kode C# atau VB.NET, kode dan markup ASP.NET, skrip pembuatan, atau XAML, gunakan pemfaktoran ulang ini untuk memastikan bahwa operasi penghapusan aman.
