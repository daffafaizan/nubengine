Elemen Contoh HTML () merender teks antara tag awal dan akhir tanpa menafsirkan HTML di antaranya dan menggunakan font monospasi. Spesifikasi HTML2 merekomendasikan bahwa itu harus dirender cukup lebar untuk memungkinkan 80 karakter per baris. Elemen asic adalah tulang punggung dokumen HTML apa pun. Anda akan melihat elemen-elemen ini dalam kode sumber untuk semua halaman web setelah deklarasi doctype pada baris pertama halaman. Doctype menentukan versi (X)HTML mana yang digunakan halaman tersebut.
