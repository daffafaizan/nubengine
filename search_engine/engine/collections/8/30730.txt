Ada tiga cara untuk membuat server virtual: virtualisasi penuh, virtualisasi para dan virtualisasi tingkat OS. Mereka semua berbagi beberapa sifat umum. Server fisik disebut host. Server virtual disebut tamu. Server virtual berperilaku seperti mesin fisik. Setiap sistem menggunakan pendekatan yang berbeda untuk mengalokasikan sumber daya server fisik untuk kebutuhan server virtual. Virtualisasi penuh menggunakan jenis perangkat lunak khusus yang disebut hypervisor.
