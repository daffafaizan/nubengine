Sebelum mencoba memadamkan api, Anda harus tahu apa yang menyala untuk memilih alat pemadam api yang tepat. Alat pemadam api kelas ABC dapat digunakan untuk berbagai jenis kebakaran dan merupakan pilihan yang baik untuk penggunaan rumah tangga. Pemadam api Kelas ABC dapat digunakan pada tiga jenis kebakaran yang berbeda: Kelas A (mudah terbakar biasa seperti kayu atau kertas), Kelas B (kebakaran cairan yang mudah terbakar seperti minyak atau bensin) atau Kelas C (kebakaran listrik). Alat pemadam api kelas ABC harus diberi label dengan jelas.
