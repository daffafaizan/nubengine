Microsoft Internet Explorer. Sering disingkat IE atau MSIE, Microsoft Internet Explorer adalah browser Internet, yang memungkinkan pengguna komputer untuk melihat halaman web. Pengguna juga dapat menggunakan Internet Explorer untuk mendengarkan dan menonton konten streaming, mengakses perbankan online, melakukan pembelian melalui Internet, dan banyak lagi. Internet Explorer pertama kali diperkenalkan pada 16 Agustus 1995 oleh Microsoft sebagai Internet Explorer 1.0 dan disertakan dengan Microsoft Windows 95.
