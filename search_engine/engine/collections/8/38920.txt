Periksa penggunaan RAM PC Anda dengan membuka Windows Task Manager. Tekan. (Pengguna Vista harus mengklik) dan mengklik tab. Jika angka di sebelah 'Tersedia' (atau 'Gratis' di Vista) di bawah 'Memori Fisik' berada di sekitar nol dan PC Anda tampak lamban, Anda memerlukan lebih banyak RAM. Pengelola Tugas Windows dapat memberi tahu Anda jika mesin Anda membutuhkan lebih banyak RAM. Windows Task Manager dapat memberi tahu Anda jika mesin Anda membutuhkan lebih banyak RAM.
