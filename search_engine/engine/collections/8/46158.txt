Untuk mem-boot dari perangkat, seperti drive USB atau CD, pilih Gunakan perangkat. Kemudian pilih USB UEFI dan perangkat akan reboot dan mulai menginstal Windows 10 Pro dari USB stick yang dapat di-boot. Saya memiliki PC Mini Voyo, tetapi Windows crash dan saya mencoba menginstalnya kembali. Saya membuat pen drive usb dengan boot EFI dan mereka tidak dikenali di boot. Saya mencoba memasukkan EFI SHELL, tetapi mereka tidak menampilkan file. Saya mencoba menggunakan CD Boot normal dengan Windows 8.1 dan mereka juga tidak bisa boot.
