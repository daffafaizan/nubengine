Unggah File Besar di IIS. Secara default, server web IIS memungkinkan ukuran file terbatas untuk diunggah ke server web. Untuk IIS 6 dan IIS 7, ukuran unggahan file maksimum default masing-masing adalah 4 MB dan 28,6 MB. IIS 7 mengembalikan kesalahan 404 (Kesalahan HTTP 404.13 - CONTENT_LENGTH_TOO_LARGE) jika seseorang mengunggah sesuatu yang lebih besar dari 30MB. Untuk memungkinkan unggahan ukuran file yang lebih besar, diperlukan beberapa perubahan server.
