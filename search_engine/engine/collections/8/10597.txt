Menguji Mikrofon Headset Anda. Ketik perekam suara di layar Mulai lalu klik Perekam Suara di daftar hasil untuk meluncurkan aplikasi. Klik tombol Mulai Perekaman lalu bicara ke mikrofon. Setelah selesai, klik tombol Berhenti Merekam dan simpan file audio di folder mana pun. Putar file audio menggunakan pemutar media, seperti Windows Media Player, Winamp atau BSPlayer. Jika Anda dapat mendengar suara Anda, mikrofon Anda berfungsi dengan baik. Jika mikrofon rusak, atau jika Anda tidak menginstal driver yang benar, Anda dapat memperbaiki masalah sebelum memulai panggilan. Windows memiliki perekam suara bawaan yang dapat Anda gunakan untuk menguji mikrofon. Jika perekam menangkap suara Anda, mikrofon Anda berfungsi dengan baik.
