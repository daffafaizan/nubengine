Menyiapkan Server LAN tanpa Hamachi! Server terraria sangat mudah diatur. Dalam panduan ini, saya akan menjelaskan kepada Anda, dengan gambar, cara mengatur yang cepat dan mudah. 1. Untuk mengatur server LAN, yaitu untuk bermain di jaringan Anda sendiri, Anda mungkin tidak memerlukan perangkat lunak lain selain terraria. 2. Untuk membuka server terraria, cukup buka direktori terraria Steam Anda. Secara default, itu terletak di C:\Program Files (x86)\Steam\SteamApps\common\Terraria.
