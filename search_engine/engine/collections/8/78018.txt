Gunung Api Komposit. Pengantar. Sebuah gunung berapi komposit juga dikenal sebagai stratovolcano. Gunung berapi ini membentuk pegunungan tinggi berbentuk kerucut. Sebagian besar gunung berapi aktif di Bumi telah membentuk cincin di sekitar Samudra Pasifik. Ada 452 stratovolcano di Cincin Api Pasifik. Lava kurang cair dan membentuk aliran lava tebal di sisi gunung berapi. Terkadang lava membentuk gunung berapi kubah kecil di atas lubang di kawah gunung berapi strato atau di sisinya yang curam. Letusan gunung api stato. Stratovolcano meletus lebih jarang daripada gunung berapi perisai yang terbentuk di atas hot spot.
