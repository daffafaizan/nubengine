Penampil AnyCAD. AnyCAD Viewer adalah perangkat lunak penampil CAD gratis untuk komputer Anda. Dengan bantuan freeware ini Anda dapat dengan mudah melihat file CAD dalam berbagai format seperti .iges, .3ds, .mdl, .obj, .md2, .stl, .dxf, .fbx, .step dll. Anda dapat mencoba freeware ini untuk membuka dan melihat file CAD dalam berbagai format.
