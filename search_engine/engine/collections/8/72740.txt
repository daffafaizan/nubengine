Saat mencoba mengunggah file lebih dari 200KB di IIS 6, file mungkin tidak pernah diunggah dan Anda mendapatkan kesalahan atau dikirim ke layar kosong. Secara default, server Windows 2003 membatasi unggahan file hingga berukuran sekitar 200KB. Untuk mengatasi batas ini Anda harus mengedit file metabase.xml IIS. Sebelum Anda dapat mengedit file metabase.xml, Anda harus memberi tahu IIS untuk mengizinkan Anda mengedit file. Di IIS, klik kanan nama server dan pilih properti.
