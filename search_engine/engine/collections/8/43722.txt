Kapan saja Anda dapat meluncurkan antarmuka Web BitMeter OS untuk memeriksa penggunaan Anda: Gunakan pintasan program di desktop dan menu Mulai, atau navigasikan ke http://localhost:2605/. Di antarmuka, tab Monitor menampilkan data bandwidth real-time pada interval 1 detik. Kecepatan unggah dan unduh ditampilkan dalam bentuk grafis dan numerik. Klik ikon jam di kanan bawah untuk menggunakan utilitas stopwatch, yang memungkinkan Anda untuk mengisolasi data penggunaan selama interval pilihan Anda.
