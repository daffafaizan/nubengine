1 Simpan header server dengan file: wget -s http://www.lycos.com/ more index.html. 2 Ambil dua level pertama `wuarchive.wustl.edu', simpan ke /tmp. wget -P/tmp -l2 ftp://wuarchive.wustl.edu/. Anda ingin mengunduh semua GIF dari direktori HTTP. `wget http://host/dir/*.gif' tidak berfungsi, karena pengambilan HTTP tidak mendukung globbing. Dalam hal ini, gunakan: wget -r -l1 --no-parent -A.gif http://Host/dir/ Ini sedikit kludge, tetapi berhasil.
