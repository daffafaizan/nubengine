Jenis, Penggunaan dan Warna Alat Pemadam Api Ringan Portabel. Sebelum Anda perlu menggunakan alat pemadam api, bacalah labelnya untuk memastikan Anda benar-benar memahami jenis alat pemadam api itu dan untuk jenis kebakaran apa yang sesuai dan aman untuk digunakan. Misalnya, alat pemadam air standar tidak boleh digunakan pada listrik.
