Buat kode Android. Maven sekarang siap untuk membangun proyek. Anda dapat menjalankan beberapa tujuan siklus hidup pembangunan dengan Maven sekarang, termasuk tujuan untuk mengkompilasi kode proyek, membuat paket perpustakaan (seperti file JAR), dan menginstal perpustakaan di lokal Repositori ketergantungan Maven.
