Masukkan kata kunci powershell lalu klik atau ketuk hasil pencarian Windows PowerShell. Cara lain untuk membuka PowerShell adalah dengan menggunakan Start Menu. Luncurkan Start Menu Windows 10 dan navigasikan ke folder aplikasi Windows PowerShell. Di dalam, Anda akan menemukan pintasan untuk PowerShell dan PowerShell ISE. Jika Anda menjalankan Windows 10 versi 64 bit, Anda juga akan menemukan entri untuk versi x86 (32 bit) aplikasi PowerShell.
