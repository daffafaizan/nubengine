DtdProcessing.Ignore untuk menonaktifkan pemrosesan DTD tanpa peringatan atau pengecualian. Untuk melakukan validasi terhadap DTD, XmlReader menggunakan DTD yang ditentukan dalam deklarasi DOCTYPE dari dokumen XML. Deklarasi DOCTYPE dapat menunjuk ke DTD inline atau dapat menjadi referensi ke file DTD eksternal. Untuk memvalidasi file XML terhadap DTD: Setel properti XmlReaderSettings.DtdProcessing ke DtdProcessing.Parse. Setel properti XmlReaderSettings.ValidationType ke ValidationType.DTD. Jika DTD adalah file eksternal yang disimpan di sumber daya jaringan yang memerlukan otentikasi, berikan objek XmlResolver dengan kredensial yang diperlukan ke metode Buat.
