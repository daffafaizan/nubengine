Tunggu Implisit; Tunggu Eksplisit; Tunggu Implisit. Selenium Web Driver telah meminjam gagasan menunggu implisit dari Watir. Penantian implisit akan memberi tahu driver web untuk menunggu selama waktu tertentu sebelum mengeluarkan Pengecualian Tanpa Elemen Tersebut. Pengaturan default adalah 0. Setelah kami mengatur waktu, driver web akan menunggu waktu itu sebelum mengeluarkan pengecualian. Dalam contoh di bawah ini kami telah mendeklarasikan sebuah implisit menunggu dengan kerangka waktu 10 detik. Artinya, jika elemen tersebut tidak berada di halaman web dalam jangka waktu tersebut, maka akan menimbulkan eksepsi. Untuk mendeklarasikan menunggu implisit: Sintaks: driver.manage().timeouts().implicitlyWait(TimeOut, TimeUnit.SECONDS);
