Tidak butuh waktu lama bagi seseorang untuk bertanya bagaimana melakukan hal yang sama untuk XenApp 6.5. Ada cukup banyak perbedaan antara layar di XenApp 6.0 dan XenApp 6.5 sehingga membingungkan. Artikel ini akan menunjukkan cara menggabungkan server XenApp 6.5 tambahan ke peternakan XenApp menggunakan Microsoft SQL Server Express. Saya tidak akan membahas instalasi XenApp 6.5 dalam artikel ini. Saya menginstal XenApp 6.5 ke sepenuhnya ditambal, pada 25-Nov-2012, server Windows Server 2008 R2 SP1. Berikut adalah langkah-langkah yang saya ambil untuk server XenApp 6.5 PERTAMA yang akan menyimpan database penyimpanan data Microsoft SQL Server Express. Setelah instalasi selesai, saya klik Specify Licensing (Gambar 1). Gambar 1. Saya selalu membuat kebijakan Citrix yang berisi informasi Server Lisensi Citrix.
