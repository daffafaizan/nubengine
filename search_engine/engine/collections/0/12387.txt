1 Ikhtisar. GNU Wget adalah utilitas gratis untuk mengunduh file non-interaktif dari Web. Ini mendukung protokol HTTP, HTTPS, dan FTP, serta pengambilan melalui proxy HTTP. Bab ini adalah gambaran sebagian dari fitur-fitur Wget. 1 Wget bersifat non-interaktif, artinya dapat bekerja di latar belakang, saat pengguna tidak masuk. 2 Wget dapat mengikuti tautan di halaman HTML, XHTML, dan CSS, untuk membuat versi lokal situs web jarak jauh, sepenuhnya menciptakan kembali struktur direktori situs asli.
