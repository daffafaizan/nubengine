Mendapatkan vitamin B-12 dalam jumlah besar melalui suntikan vitamin B-12 tidak akan membahayakan kesehatan Anda. Tetapi vitamin B-12 dapat mengganggu efektivitas beberapa obat, jadi jika Anda menerima suntikan vitamin B-12, pastikan untuk memberi tahu dokter Anda. Vitamin B-12 juga ditambahkan ke beberapa makanan dan tersedia sebagai suplemen makanan. Suntikan vitamin B-12 biasanya diresepkan untuk membantu mencegah atau mengobati anemia pernisiosa dan defisiensi B-12. Beberapa klinik penurunan berat badan menawarkan suntikan vitamin B-12 sebagai bagian dari program penurunan berat badan mereka.
