File yang berisi ekstensi file .ex_ adalah file yang dapat dieksekusi terkompresi. Ketika file-file ini tidak dikompresi, mereka disimpan dengan ekstensi file .exe. File yang dapat dieksekusi EXE atau EX_ biasanya dibuat oleh program arsip instalasi. File EXE sering digunakan untuk menginstal perangkat lunak ke komputer pengguna. Terkadang file EXE dikompresi agar lebih mudah diunduh, membuat unduhan file lebih kecil. Ketika file EXE dikompresi, file tersebut diberi ekstensi file .ex_ daripada ekstensi file .exe standar.
