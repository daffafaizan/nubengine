Pembakaran dan Ventilasi Tungku Propana. Tungku gas LP harus selalu menarik udara untuk pembakaran dari luar ruang tamu suatu struktur. Udara harus diambil dari loteng atau dari luar rumah. Ini sepenuhnya untuk keselamatan dan untuk pencegahan keracunan karbon monoksida di dalam ruang tamu rumah. Pemeriksaan sederhana ini tidak bersifat teknis dan dapat menjaga tungku dan AC dalam keadaan baik ketertiban kerja serta menjaga keluarga dan rumah mereka aman. 1 Api Tungku-Nyala api kompor tungku propana harus berwarna biru. 2 Jika api jingga atau kuning terlihat selama pengoperasian, tungku perlu disetel.
