Alat Arp memungkinkan pengguna untuk melihat dan memodifikasi entri tabel ARP di komputer lokal. Perintah arp berguna untuk melihat cache ARP dan menyelesaikan masalah resolusi alamat. Entri statis dapat ditambahkan ke file ARP dengan mengeluarkan perintah arp -s < IP address > < MAC address > . Namun, menambahkan entri cache ARP statis seperti itu harus digunakan dengan hati-hati karena mudah memasukkan alamat MAC yang salah untuk alamat IP. Mendeteksi Duplikat Alamat IP Menggunakan ARP Saat memulai, Windows melakukan ARP serampangan untuk mendeteksi duplikasi dengan alamat IP-nya sendiri.
