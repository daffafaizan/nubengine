Sebuah Dokumen Keamanan sendiri tidak mengizinkan akses ke jalur prioritas. Namun, jika Anda bepergian dengan Business Select atau merupakan pemegang kartu A-List, Anda akan dapat menggunakan Enhanced Security Document Anda untuk mengakses jalur prioritas (jika tersedia).n Enhanced Security Document biasanya diterbitkan sebagai pengganti boarding pass ketika reservasi memerlukan pemrosesan manual oleh Karyawan bandara, seperti saat Anda perlu menukar tiket, memverifikasi usia, atau mengisi dokumen di bandara sebelum menerima boarding pass.
