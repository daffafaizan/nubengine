Alat pemadam api kimia kering memadamkan api terutama dengan mengganggu reaksi kimia segitiga api. Jenis alat pemadam api yang paling banyak digunakan saat ini adalah bahan kimia kering multiguna yang efektif pada kebakaran Kelas A, B, dan C. Agen ini juga bekerja dengan menciptakan penghalang antara elemen oksigen dan elemen bahan bakar pada kebakaran Kelas A.
