PowerShell Remoting memungkinkan Anda menjalankan perintah PowerShell individual atau mengakses sesi PowerShell penuh pada sistem Windows jarak jauh. Ini mirip dengan SSH untuk mengakses terminal jarak jauh pada sistem operasi lain. PowerShell dikunci secara default, jadi Anda harus mengaktifkan PowerShell Remoting sebelum menggunakannya. Mengaktifkan PowerShell Remoting. Di komputer yang ingin Anda akses dari jarak jauh, buka jendela PowerShell sebagai Administrator - klik kanan pintasan PowerShell dan pilih Jalankan sebagai Administrator. Untuk mengaktifkan PowerShell Remoting, jalankan perintah berikut (dikenal sebagai cmdlet di PowerShell): Enable-PSRemoting-Force.
