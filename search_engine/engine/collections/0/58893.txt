Dalam kebanyakan kasus, Anda hanya perlu menunjukkan boarding pass untuk naik ke pesawat. ID yang dapat diterima untuk perjalanan dari AS. Ada dua jenis persyaratan identifikasi untuk perjalanan internasional dari AS, identifikasi untuk melewati keamanan TSA, dan identifikasi untuk maskapai. Persyaratan ID TSA untuk memasuki bagian aman terminal bandara sama untuk penumpang domestik atau internasional.dapat diterima ID untuk perjalanan domestik di luar AS. Banyak negara memiliki persyaratan ID untuk penerbangan domestik yang serupa dengan persyaratan AS. Anda harus memeriksa dengan maskapai penerbangan Anda atau dengan otoritas yang sesuai di negara tersebut untuk mengetahui ID apa yang dapat diterima untuk penerbangan domestik.
