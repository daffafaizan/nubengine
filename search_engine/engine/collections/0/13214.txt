Cara menemukan alamat IP Anda di Windows (dan cara menemukan alamat IP router Anda). Di menu Mulai (atau layar mulai di Windows 8) ketik cmd.exe dan tekan Enter. Ini akan meluncurkan Command Prompt-jendela hitam seperti di bawah ini. Kami menggunakan Windows 8 di sini, tetapi pada dasarnya sama di versi Windows sebelumnya. Panduan langkah demi langkah menjelaskan cara mengetahui alamat IP perangkat PC, iPhone, iPad, atau Android Anda. Alamat IP adalah nomor yang membuat perangkat dapat diidentifikasi di jaringan.
