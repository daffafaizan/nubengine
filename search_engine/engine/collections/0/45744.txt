Mirena/menghentikan periode. Saya telah memiliki Mirena selama tiga bulan. saya mendapatkannya pada awal Juni. saya haid awal juli. itu jauh lebih ringan dari biasanya, dan saya tidak mengalami kram. setelah periode saya selesai, saya melihat dan terus selama sekitar tiga minggu. aku baik-baik saja sejak itu. sekarang akhir Agustus (hampir dua bulan sejak saya terakhir haid).
