Gambar PNG juga tidak didukung dengan baik oleh ponsel dan ponsel berfitur lama. Kami menggunakan PNG untuk file apa pun yang membutuhkan transparansi. Kami juga menggunakan PNG-8 untuk file apa pun yang cocok sebagai GIF, menggunakan format PNG ini sebagai gantinya. Gambar SVG. SVG adalah singkatan dari Scalable Vector Graphic. Berbeda dengan format berbasis raster yang ditemukan di JPG, GIF, dan PNG, file-file ini menggunakan vektor untuk membuat file yang sangat kecil yang dapat dirender pada ukuran apa pun tanpa kehilangan kualitas peningkatan ukuran file.
