Gaji Rata-Rata. Gaji rata-rata pemain NFL bekerja jauh lebih sedikit jika tidak mempertimbangkan penerima teratas, tetapi para pemain ini masing-masing masih menghasilkan ratusan ribu dolar. Bloomberg Businessweek mengatakan gaji rata-rata pemain NFL pada 2011 adalah $1,9 juta. Gaji rata-rata adalah $770.000. Gaji rata-rata seorang pemain NFL bekerja jauh lebih sedikit jika tidak mempertimbangkan penerima tertinggi, tetapi para pemain ini masing-masing masih menghasilkan ratusan ribu dolar.
