Langkah 4: Membaca Tag Pemadam Kelas. Setiap alat pemadam kebakaran, baik untuk penggunaan di rumah atau komersial, harus memiliki label atau label dengan kelas alat pemadam yang terpampang dengan jelas. Label atau tag ini akan terbaca Kelas A, Kelas B, Kelas C, Kelas D atau Kelas ABC.. Penunjukan ini memberi tahu Anda jenis api apa yang dapat digunakan alat pemadam. Menggunakan jenis alat pemadam yang salah, seperti alat pemadam api kertas dan kayu pada kebakaran gas atau listrik, dapat membunuh Anda atau mempercepat kebakaran. Langkah 1: Temukan Alat Pemadam Api Anda. Jika Anda bekerja untuk bisnis, secara hukum, alat pemadam kebakaran diperlukan untuk bangunan. Cari setiap pemadam. Sebagian besar terletak di dekat atau di dekat pintu keluar, pintu, atau di area dekat tempat kemungkinan besar terjadinya kebakaran, seperti di dekat kompor, pemanas, atau area pengumpulan sampah.
