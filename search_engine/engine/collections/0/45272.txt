File batch untuk menghapus file pengguna sementara. File batch Windows adalah jenis file teks, yang berisi serangkaian perintah untuk sistem operasi komputer. Anda tidak boleh membuka file batch pada hard disk Anda yang tidak dikenal karena berisi perintah yang dapat dieksekusi. Ekstensi file untuk file batch adalah .BAT. File Temp (File sementara) dibuat oleh program sistem pada dasarnya ketika program tidak bisa mendapatkan tingkat yang sesuai memori untuk menyelesaikan tugas.
