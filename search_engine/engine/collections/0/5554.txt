Alat Pemadam Api Halon Alat pemadam api halon adalah jenis khusus alat pemadam api yang mengandung gas yang mengganggu reaksi kimia yang terjadi ketika bahan bakar terbakar. Alat pemadam api jenis ini sering digunakan untuk melindungi peralatan listrik yang berharga karena tidak meninggalkan residu.
