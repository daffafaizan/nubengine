Program yang Tidak Kompatibel dan Jalankan Sendiri. Saat Anda menentukan program bersamaan, Anda dapat membuat daftar program yang Anda inginkan agar tidak kompatibel. Anda juga dapat membuat daftar program sebagai tidak kompatibel dengan dirinya sendiri, yang berarti bahwa dua contoh program tidak dapat berjalan secara bersamaan. Anda juga dapat membuat program tidak kompatibel dengan semua program bersamaan lainnya dengan mendefinisikan program untuk dijalankan sendiri. Anda mendefinisikan program konkuren untuk dijalankan sendiri atau tidak kompatibel dengan program konkuren tertentu dengan mengedit definisi program konkuren menggunakan jendela Concurrent Programs.
