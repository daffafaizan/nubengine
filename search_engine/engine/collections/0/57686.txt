Beberapa printer juga memungkinkan Anda memberi nama file Anda sebelum memindai dokumen Anda, tetapi Anda juga dapat melakukannya setelah dokumen diubah menjadi file pdf. Anda cukup klik kanan pada nama dokumen dan pilih rename untuk membuat perubahan. Anda juga dapat pergi ke file dan pilih rename juga.mages.google.com. Memindai dokumen menjadi pdf sangat sederhana dengan teknologi saat ini. Bahkan tantangan teknologi dapat memindai dokumen ke dalam format pdf dalam waktu singkat. Anda juga dapat mengirim file pdf melalui email jika perlu.
