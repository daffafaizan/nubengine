Menggunakan sub-kueri berkorelasi di SQL Server. Memiliki kemampuan untuk menggunakan sub-kueri yang berkorelasi akan memungkinkan Anda untuk memecahkan masalah yang tidak dapat dilakukan oleh pengembang basis data lainnya. Tim Chapman membahas bagaimana menggunakan konstruksi pemrograman yang sangat berguna ini. Sub-query adalah pernyataan SQL Server yang disematkan di dalam pernyataan SQL Server lainnya. Mesin database memperlakukan sub-kueri sebagai tabel virtual untuk eksekusi kueri. Sub-kueri dapat digunakan sebagai tabel dalam pernyataan gabungan, sebagai nilai tunggal dalam pernyataan pilih, dalam klausa where dari kueri SQL Server, dalam klausa memiliki dari kueri SQL Server, atau digabungkan dalam pernyataan manipulasi data . Eksekusi sub-kueri tergantung pada tingkat kueri yang bersarang.
