Ada kemungkinan versi JRE berbeda dari versi JDK. Anda dapat menentukan versi JDK mana yang default dengan mengetikkan versi java di jendela Terminal. Jika versi yang diinstal adalah 7u55, Anda akan melihat string yang menyertakan teks Versi 1.7.0_55.ava di Panel Kontrol Java-Windows dan Mac OS X. Versi Java dapat ditemukan di Panel Kontrol Java. 1 Temukan Panel Kontrol Java di Windows. 2 Temukan Java Control Panel di Mac OS X. 3 Di bawah tab General di Java Control Panel, versi tersedia melalui bagian About.
