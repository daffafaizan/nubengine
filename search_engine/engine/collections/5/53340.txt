Beranda > Cara Membunuh Semut Api > Pengobatan Rumahan. Banyak pengobatan rumah semut api telah diusulkan dan digunakan. Namun, banyak produk semut api di pasaran seaman atau lebih aman daripada pengobatan rumahan. Seringkali, pengobatan rumahan tidak membunuh seluruh koloni. Untuk benar-benar menghilangkan koloni, perlu untuk membunuh ratu bertelur. Untuk cara yang aman dan efektif untuk mengendalikan semut api di pekarangan rumah, lihat Pengendalian Semut Api: Metode Dua Langkah dan Pendekatan Lainnya.
