Ketika kita menggunakan metode start maka sebuah thread baru dibuat dan kemudian kode di dalam metode run akan dieksekusi untuk setiap Thread baru. Penggunaan metode start membuat dua tumpukan untuk setiap utas, Tumpukan dan tumpukan asli.
