Handel - Musik untuk Kembang Api Kerajaan Musik untuk Kembang Api Kerajaan Handel adalah musik yang cocok untuk seorang raja - George II, sebenarnya, yang menghadiri pertunjukan pertama di Sungai Thames pada tahun 1749. Handel menggubah Musiknya untuk Kembang Api Kerajaan saat bekerja untuk George II pada tahun 1749.
