16 Jawaban 16. Wget untuk Windows seharusnya berfungsi. GNU Wget adalah utilitas jaringan gratis untuk mengambil file dari World Wide Web menggunakan HTTP dan FTP, dua protokol Internet yang paling banyak digunakan. Ini bekerja secara non-interaktif, sehingga memungkinkan pekerjaan di latar belakang, setelah logout.
