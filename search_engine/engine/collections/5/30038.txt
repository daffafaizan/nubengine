HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Authentication\Credential Providers. Pengaturan kebijakan ini memungkinkan administrator untuk menetapkan penyedia kredensial tertentu sebagai penyedia kredensial default. Jika Anda mengaktifkan pengaturan kebijakan ini, penyedia kredensial yang ditentukan dipilih pada ubin pengguna lain. Tekan kombinasi Tombol Windows + R, ketik regedit di kotak dialog Run dan tekan Enter untuk membuka Registry Editor. 2. Arahkan ke sini: HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Authentication\Credential Providers. Daftar penyedia kredensial terdaftar dan GUID mereka dapat ditemukan di sini.
