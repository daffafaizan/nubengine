Ada banyak jenis virtualisasi. Definisi virtualisasi yang paling umum adalah mengambil server dan membuat komputer virtual di dalamnya untuk menjalankan perangkat lunak. Virtualisasi server memungkinkan banyak salinan Windows dan/atau Linux dijalankan dari server fisik yang sama. Satu server fisik dapat menjalankan server virtual sebanyak yang dapat didukung oleh perangkat keras.
