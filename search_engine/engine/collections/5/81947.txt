Alat Pemadam Api ABC Alat Pemadam Api ABC adalah rumah atau tempat kerja di mana kebakaran dapat terjadi akibat pembakaran sampah, kayu, kertas, cairan, gas, pelarut & peralatan listrik. Alat Pemadam Api ABC sangat serbaguna. Mereka sering menjadi pilihan ideal karena mereka mampu memadamkan berbagai jenis api. Mereka menggunakan monoammonium phosphate yang merupakan bahan kimia kering yang mampu memadamkan api dengan cepat.
