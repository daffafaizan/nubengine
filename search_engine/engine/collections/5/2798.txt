Ketika saya pergi untuk menghapusnya, Windows tidak mengizinkan saya menghapus file Thumbs.db. Saya dapat menghapus semua yang ada di folder (file kata dan file gambar) tetapi tidak dapat menghapus file Thumbs.db. Windows terus memberi tahu saya bahwa itu tidak dapat menghapus file Thumbs.db dan bahwa program atau orang lain sedang menggunakannya. Hanya saja tidak ada program lain yang terbuka dan tidak ada orang lain yang masuk.
