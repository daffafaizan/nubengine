Kami menjelaskan apa itu file PNG dan merekomendasikan perangkat lunak yang kami tahu dapat membuka atau mengonversi file PNG Anda. PNG adalah singkatan dari Portable Network Graphic. Format file PNG menawarkan format file lossless terkompresi untuk gambar bitmap grafis. File PNG sangat mirip dengan file GIF. Mereka berisi bitmap dan menawarkan algoritma kompresi lossless tetapi mereka tidak mengandung batasan hak cipta.
