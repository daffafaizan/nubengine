Warna asli, file kecil. Format file PNG (Portable Network Graphics) dianggap, dan dibuat, sebagai penerus gratis dan sumber terbuka untuk format file GIF. Format file PNG mendukung warna asli (16 juta warna) sedangkan format file GIF hanya memungkinkan 256 warna. PNG unggul jika gambar memiliki area warna seragam yang luas. Format lossless PNG paling cocok untuk mengedit gambar, dan format lossy seperti JPG paling baik untuk distribusi akhir gambar jenis fotografi karena ukuran file yang lebih kecil.
