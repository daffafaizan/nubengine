Untuk memperbarui file .prf secara manual. 1 Buka file .prf dengan menggunakan editor teks seperti Notepad. 2 Lakukan perubahan atau penambahan. Dengan mengedit file .prf secara manual, Anda dapat menambahkan layanan MAPI apa pun yang didukung oleh Outlook 2010. 3 Setelah Anda menyelesaikan perubahan, simpan file tersebut.
