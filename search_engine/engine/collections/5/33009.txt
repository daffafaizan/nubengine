Referensi Meja All-In-One Java Untuk Dummies, Edisi ke-2. Java adalah bahasa pemrograman dalam tradisi C dan C++. Akibatnya, jika Anda memiliki pengalaman dengan C atau C++, Anda akan sering menemukan diri Anda di wilayah yang akrab saat Anda mempelajari berbagai fitur Java. Namun, Java berbeda dari bahasa pemrograman lain dalam beberapa hal yang signifikan.
