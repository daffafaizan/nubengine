File dengan ekstensi file XAML (diucapkan sebagai zammel) adalah file Bahasa Markup Aplikasi yang Dapat Diperluas, dibuat menggunakan bahasa markup Microsoft dengan nama yang sama. XAML adalah bahasa berbasis XML, jadi file .XAML pada dasarnya hanyalah file teks. Mirip dengan bagaimana file HTML digunakan untuk mewakili halaman web, file XAML menjelaskan elemen antarmuka pengguna dalam aplikasi perangkat lunak untuk aplikasi Windows Phone, aplikasi Windows Store, dan banyak lagi.
