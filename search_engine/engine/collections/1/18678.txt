Secara umum, alat pemadam kebakaran terbaik untuk laboratorium adalah: 1 Alat pemadam api karbon dioksida (CO2) Alat pemadam api CO2 akan memadamkan kebakaran yang disebabkan oleh listrik atau cairan yang mudah terbakar. Ini adalah alat pemadam api terbaik untuk banyak laboratorium karena tidak meninggalkan residu yang merusak yang dapat membahayakan peralatan elektronik.
