Debug file dukungan (Power CTrace). TRC adalah ekstensi file untuk file debug yang digunakan oleh perangkat lunak database Oracle. TRC adalah singkatan dari TRACe. File TRC berisi log peristiwa yang menjelaskan transaksi apa yang telah terjadi dalam database. File TRC biasanya digunakan saat mendiagnosis kesalahan atau untuk men-debug database.
