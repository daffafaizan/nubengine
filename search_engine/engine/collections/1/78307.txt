File data untuk setiap aplikasi disimpan dalam folder %USERPROFILE%\AppData\Local\Packages\{PackageId} , yang biasanya adalah C:\Users\{UserName}\AppData\Local\Packages\{PackageId} , di mana {UserName } sesuai dengan nama pengguna Windows dan {PackageId} sesuai dengan pengidentifikasi paket aplikasi Windows Store. Yang cukup menarik, aplikasi Store tidak disimpan dalam folder yang sama dengan semua aplikasi Windows Store lainnya, dan terletak di %windir%\WinStore , yang biasanya adalah C:\Windows\WinStore. Catatan ini telah diposting di Windows 8 dan menandai aplikasi Metro, Windows 8, Windows Store, Windows Store.
