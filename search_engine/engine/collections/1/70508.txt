File Riwayat Harga Quicken disimpan dalam format QPH dan ditempel dengan ekstensi QPH, dan digunakan oleh perangkat lunak Quicken. File QPH ini umumnya diklasifikasikan sebagai file data yang berisi catatan daftar harga dari waktu ke waktu yang diunduh atau dimasukkan secara manual untuk investasi di akun Quicken.
