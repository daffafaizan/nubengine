Pengontrol Host USB. USB Host Controller adalah perangkat keras baik pada motherboard komputer atau kartu PCI. Ini menyediakan antarmuka untuk mentransfer aliran data antara komputer host dan perangkat USB. Komputer host mungkin memiliki satu atau beberapa pengontrol host USB dengan jenis antarmuka yang berbeda. Pengontrol Host SB.
