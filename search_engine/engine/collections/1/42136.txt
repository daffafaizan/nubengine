Menambahkan tabel ke halaman Web. Untuk menambahkan tabel ke halaman Web di situs Microsoft Office SharePoint Server 2007, Anda menggunakan Editor HTML. Anda bisa menggunakan Editor HTML untuk menambahkan tabel ke kontrol bidang Konten Halaman atau ke Komponen Web Editor Konten. pena editor di kontrol bidang Konten Halaman. Pada halaman yang ingin Anda tambahkan tabelnya, klik Perlihatkan Bilah Alat Pengeditan Halaman pada menu Tindakan Situs. Pada toolbar Pengeditan Halaman, klik Edit Halaman. Di kontrol bidang Konten Halaman, klik Edit Konten. Bilah alat Editor HTML muncul.
