Untuk menyambung ke komputer jarak jauh: Setelah Anda membuat sambungan Desktop Jarak Jauh, desktop jarak jauh Anda akan ditampilkan di jendelanya sendiri. Anda dapat menggunakan keyboard dan mouse dari host lokal untuk mengontrol komputer jarak jauh. Anda dapat memperoleh akses ke drive disk Anda di komputer lokal selama sesi Remote Desktop.
