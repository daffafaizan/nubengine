Pilih Adobe PDF dari menu printer. Klik tombol Properties (atau Preferences) untuk menyesuaikan pengaturan printer Adobe PDF. (Dalam beberapa aplikasi, Anda mungkin perlu mengklik Setup di kotak dialog Print untuk membuka daftar printer, lalu klik Properties atau Preferences.). Di kotak dialog Cetak, klik OK.y default, PDF Anda disimpan dengan nama file yang sama dan ekstensi .pdf. Ke atas. Preferensi pencetakan berlaku untuk semua aplikasi yang menggunakan printer Adobe PDF, kecuali jika Anda mengubah pengaturan dalam aplikasi pembuatan dengan menggunakan menu Pengaturan Halaman, Pengaturan Dokumen, atau Cetak.
