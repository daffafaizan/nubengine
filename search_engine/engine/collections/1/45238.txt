Untuk panduan lebih lanjut tentang komputasi awan, lihat Publikasi Khusus NIST 800-145 - Definisi Komputasi Awan NIST. Model penyebaran komputasi awan: Teknologi awan dapat digunakan sebagai awan pribadi, komunitas, publik, atau hibrida. Lihat NIST SP 800-145, Definisi NIST Cloud Computing untuk panduan lebih lanjut. Cloud Computing Governance Board (CCGB): CCGB memberikan rekomendasi kepada pejabat yang berwenang (AO). Dewan akan diketuai oleh PD/CIO untuk IRM dan akan meninjau semua proyek dan program berbasis cloud selama fase perencanaan (5-FAH-8 H-350).
