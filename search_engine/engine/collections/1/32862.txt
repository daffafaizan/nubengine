Host RDS adalah komputer server yang menghosting aplikasi dan sesi desktop untuk akses jarak jauh. Host RDS dapat berupa mesin virtual atau server fisik. Dalam tampilan, host RDS adalah server yang memiliki peran Layanan Desktop Jarak Jauh Microsoft, layanan Host sesi Desktop Jarak Jauh Microsoft, dan Agen Tampilan yang diinstal. Layanan Desktop Jarak Jauh sebelumnya dikenal sebagai Layanan Terminal. Layanan Host Sesi Desktop Jarak Jauh memungkinkan server untuk meng-host aplikasi dan sesi desktop jarak jauh. Dengan View Agent yang diinstal pada host RDS, pengguna dapat terhubung ke aplikasi dan sesi desktop dengan menggunakan protokol tampilan PCoIP.
