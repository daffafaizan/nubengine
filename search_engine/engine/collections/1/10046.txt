Cara Membuka File MDI. MDI, kependekan dari Microsoft Document Imaging, adalah format file milik Microsoft. File MDI dapat dilihat di OneNote atau MDI Viewer; keduanya disertakan dalam versi Microsoft Office sebelum Office 2010.OneNote diinstal secara default dengan program Office lainnya, tetapi aplikasi MDI Viewer harus diaktifkan sebelum dapat digunakan untuk melihat file MDI. File MDI berisi gambar yang serupa dalam kualitas ke TIFF tetapi membutuhkan lebih sedikit ruang pada hard drive.neNote diinstal secara default dengan program Office lainnya, tetapi aplikasi MDI Viewer harus diaktifkan sebelum dapat digunakan untuk melihat file MDI.
