Memilih preset adalah cara termudah untuk mengatur opsi untuk file PDF Photoshop Anda. Setelah Anda memilih preset, klik Save PDF untuk menghasilkan file PDF Photoshop Anda. Jika Anda ingin menambahkan opsi keamanan atau menyempurnakan opsi penyimpanan untuk PDF, ikuti langkah selanjutnya dalam prosedur ini.ave As, lalu pilih Photoshop PDF dari menu Format. Anda dapat memilih opsi Warna jika Anda ingin menyematkan profil warna atau menggunakan profil yang ditentukan dengan perintah Proof Setup. Anda juga dapat menyertakan lapisan, catatan, warna titik, atau saluran alfa. Klik Simpan.
